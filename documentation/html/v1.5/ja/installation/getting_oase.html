2.1. Exastro OASE の入手¶

Exastro OASE の入手方法は下記の2つの方法があります。

  • リリース資材

  • コンテナ

リリース資材を利用する場合は、 リリースページ からファイルをダウンロードして下さい。
コンテナを利用する場合は、Docker Hub からイメージの取得を行って下さい。
