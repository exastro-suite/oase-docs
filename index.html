Exastro
Operation Autonomy
Support Engine

Exastro OASE is a Open-source software
that supports autonomination, efficiency,
and effortlessness for system operation.

  • Description

Exastro OASE solves 3 problems during system operation.

  • Problem breakdown and solution finding relys on experts.
    The system will determine the solution automatically by the rules that has already been set.
  • Operation errors according to complicated manual work.
    Eliminate human errors by linking with work automation software and executing troubleshooting automatically.
  • Expansion of service effect due to the delay of problem recovery.
    The effect is minimized since solution is applied right after problem examination.

Exastro OASE overview

"6 Features" of Exastro OASE

  1. 　Confirm rule list in staging environment
  2. 　Link with multiple monitoring software and automation software
  3. 　Define the rules without coding
  4. 　Control the number of action execution
  5. 　Manage multiple rule list
  6. 　Role-based access control
