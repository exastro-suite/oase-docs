Exastro Operation Autonomy Support Engine online installation

Here is the installation manual for users to quickly install Operation Autonomy Support Engine to their devices. The installation is compitable with RedHat Linux distributions and Internet access is required.

After installation, users can refer to Operation Autonomy Support Engine quickstart manual in below to start to experience the amazing interface of Operation Autonomy Support Engine.

Getting started with Exastro Operation Autonomy Support Engine

Exastro Operation Autonomy Support Engine quickstart

This is a quickstart manual for users to familiarize with the interface of Operation Autonomy Support Engine.

Operation Autonomy Support Engine can associate with several software, users can experience association with mail system which is easy to setup among the software by following this manual.

In order to provide a smooth introduction, the most important functions provided by Operation Autonomy Support Engine including "Monitoring adapter" and "Action excution associated with ITA" is not mentioned in this manual.
